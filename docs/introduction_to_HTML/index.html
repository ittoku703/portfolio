HTML要素の分解

HTMLの要素は主に、4つの部分から構成されています。

  • 開始タグ（Opning tag）
  • 終了タグ（Closing tag）
  • コンテンツ（Content）
  • エレメント（Element）
開始タグ（Opning tag）
この部分は「要素の開始」を意味しています。
終了タグ（Closing tag）
この部分は「要素の終了」を意味しています。
コンテンツ（Content）
この部分は開始タグと終了タグの間の部分で、テキストや要素を入れ子にしたりします。
エレメント（Element）
この部分は、開始タグ、終了タグ、コンテンツを含むすべての総称です。

以下は使用例となります

HTML
		
<p>Hello World!<\p>

ブロック要素とインライン要素

ブロック要素とインライン要素はHTMLにおける重要なカテゴリーの一つです。では、それらはどういったものなのか見ていきましょう。

ブロック要素

ブロック要素は、Webページ上で視認できるブロックを形成します。
ブロックは新たな行におけるコンテンツとして表示され、そのブロックの後に来る新たなるコンテンツも新しい行で表示される。

段落・リスト・ナビゲーションメニュー・フッターなどに使用される傾向にある

インライン要素

インライン要素は、ブロックレベル要素の中に包含され、なおかつ段落全体やコンテンツのグループではなく、ドキュメントの内容の小さな部分だけを囲む要素である
また新たな行としては表示されない。

a, em, strongなどの強調要素のように、テキスト段落の中で表示される

空要素

全ての要素が開始タグ・コンテンツ・終了タグのパターンに従っているわけではありません。
から要素は「開始タグ」のみで完結する要素のことです。
